# AI Tetris
AI implemented over a simple Tetris game built with Pygame.

## How to Play
AI Tetris can be downloaded and played in a few simple steps.

#### 1. Clone the Repository
```sh
git clone https://github.com/gumdropsteve/ai-tetris
cd ai-tetris
```

#### 2. Install Pygame
```sh
pip install pygame
```

#### 3. Run the Game
```sh
python tetris.py
```
